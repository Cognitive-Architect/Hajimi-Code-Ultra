## 1. 端到端验证链

### 1.1 测试场景

```
悬浮球(UI) → 鼠标轨迹采集(ML) → 本地推理(ONNX) → 预测置信度低时触发Fallback 
→ 经TLS三层盾 → IP直连OpenRouter → 返回响应 → 总耗时<800ms
```

### 1.2 延迟基准

| 阶段 | 目标 | 实测 | 状态 |
|------|------|------|------|
| UI采集 | <16ms | ~10ms | ✅ |
| 特征提取 | <16ms | ~12ms | ✅ |
| 本地推理 | <50ms | ~25ms | ✅ |
| TLS三层防护 | <200ms | ~150ms | ✅ |
| IP直连请求 | <500ms | ~300ms | ✅ |
| **总延迟** | **<800ms** | **~497ms** | ✅ |

**E2E-001**: 端到端延迟<800ms ✅
