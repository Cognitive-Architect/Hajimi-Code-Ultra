# Alice ML 数据隐私政策

> **文档编号**: HAJIMI-ALICE-PRIV-001  
> **作者**: 💛 Soyorin (PM) - B-03/09  
> **生效日期**: 2026-02-17

---

## 1. 核心原则

### 1.1 本地优先 (Local First)

**原始鼠标轨迹数据永不离开设备。**

- ✅ 坐标 (x,y) 仅存储于本地
- ✅ 所有特征提取在端侧完成
- ✅ 云端仅接收 12维归一化特征向量

### 1.2 数据最小化

- 仅收集行为预测必需的轨迹数据
- 不关联用户身份
- 自动清理过期数据（默认7天）

### 1.3 用户控制

- 一键清除所有历史数据
- 随时退出数据收集
- 透明的数据使用说明

---

## 2. 数据处理流程

```
[鼠标移动] 
    ↓
[本地采集] ──▶ 原始轨迹 (x,y,t)
    ↓
[特征提取] ──▶ 12维特征向量 (脱敏)
    ↓
[本地推理] ◀── 特征向量 ──▶ [云端Fallback*]
    ↓
[行为预测] ──▶ Alice响应
    ↓
[反馈收集] ──▶ 本地存储 (加密)
```

\* 云端Fallback仅在用户明确授权且本地推理失败时使用

---

## 3. 数据分类

| 类别 | 内容 | 存储位置 | 保留期限 |
|------|------|----------|----------|
| **原始轨迹** | (x,y,t) 坐标 | 设备内存 | 会话结束即销毁 |
| **特征向量** | 12维归一化特征 | IndexedDB | 7天 |
| **预测结果** | 行为分类 | IndexedDB | 7天 |
| **用户反馈** |  thumbs up/down | IndexedDB | 30天 |

## 4. 技术保护措施

### 4.1 本地存储加密

```typescript
// AES-256-GCM 加密
const encrypted = await crypto.subtle.encrypt(
  { name: 'AES-GCM', iv },
  key,
  data
);
```

### 4.2 差分隐私

训练数据聚合时注入噪声：
```
published_data = true_mean + laplace_noise(ε)
```

### 4.3 k-匿名性

云端数据聚合满足 k=5 匿名性要求

## 5. 用户权利

### 5.1 访问权

用户可通过开发者工具查看本地存储的所有数据

### 5.2 删除权

```typescript
// 一键清除
alicePrivacyGuard.clearAllData();
```

### 5.3 退出权

用户可随时关闭 Alice ML 功能，回退到纯启发式模式
